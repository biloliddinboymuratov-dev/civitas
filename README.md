# Civitas Global

A responsive static website for Civitas Global designed for a white, professional civic organization style.

## Sections

- Home
- News
- Blog
- Achievements
- Events
- Press
- Contact
- Admin Panel

## Files

- `index.html` — main website structure
- `admin.html` — admin panel for managing content locally
- `styles.css` — styling and responsive layout
- `script.js` — contact form and saved content logic
- `admin.js` — admin panel behavior and local storage management

## Open locally

Open `index.html` in your browser to view the site.

To access the admin tools, open `admin.html` in your browser.
